# CIFAR-10 Image Classifier

Это проект для классификации изображений из набора данных CIFAR-10 с помощью сверточной нейронной сети (CNN) на PyTorch и веб-приложения на Streamlit.

## Описание
Проект умеет:
- Обучать модель CNN для классификации 10 классов CIFAR-10 (`plane`, `car`, `bird`, `cat`, `deer`, `dog`, `frog`, `horse`, `ship`, `truck`)
- Загружать модель и использовать её через веб-приложение на Streamlit
- Позволять пользователю загружать своё изображение и получать предсказание класса

## Файлы проекта
- `cifar10_training.ipynb` — ноутбук с обучением модели
- `streamlit_app.py` — веб-приложение
- `cifar_net.pth` — сохранённая модель
- `requirements.txt` — список зависимостей Python
- `README.md` — описание проекта
